# 📌 Diretrizes de Conteúdo: Pinterest

O Pinterest é um **mecanismo de busca visual**. Diferente de redes sociais tradicionais, o usuário aqui está procurando **dicas de saúde, guias práticos, ideias de bem-estar e soluções para problemas do cotidiano pet**. Ele busca salvar ideias (Pins) em pastas para consultar depois.

---

## 🎯 1. Principais Termos de Busca do Tutor (Pinterest)
Os títulos e descrições dos nossos Pins devem conter os termos que os tutores pesquisam ativamente na barra de busca do Pinterest:

* **Buscas sobre Cães:**
  * `como dar remedio para cachorro`, `como saber se cachorro esta com dor`
  * `tabela de vacina para cachorro filhote`, `comportamento canino dicas`
  * `alimentacao natural para caes receitas`, `coceira em cachorro o que fazer`
* **Buscas sobre Gatos:**
  * `como dar comprimido para gato facil`, `comportamento de gatos significado`
  * `sintomas de dor em gato`, `brinquedos para gatos caseiro`
* **Buscas Gerais:**
  * `plantas toxicas para cachorros e gatos`, `como tirar cheiro de xixi de pet`

## 🎨 2. Design Visual dos Pins (Template Otimiza V5)
Para atrair cliques no mar de imagens do Pinterest, nossos Pins devem seguir o padrão de identidade visual que desenvolvemos no script `image_creator.py`:

1. **Formato:** Proporção vertical `9:16` (Resolução recomendada: `1000px x 1500px`).
2. **Identidade Visual:** Fundo com gradiente nas cores da marca (Roxo Intenso Otimiza `#470C51` mesclando com `#6C0C51`).
3. **Banner de Texto (Overlay):** Um retângulo sólido no rodapé (opacidade 90%) contendo o título em caixa alta (letras brancas e grandes em Arial Bold) para facilitar a leitura rápida na tela do celular.
4. **Tag de Marca:** Indicação discreta `OTIMIZA FARMAVET | CATEGORIA` no rodapé da caixa.

## 🚫 4. Regras do Brand System
* **Proibido diminutivos:** Não use `cachorrinho`, `gatinho`, `bichinho`. Use `cachorro`, `gato` ou `pet`.
* **Sem gatilho de medo:** Não use fotos trágicas ou títulos assustadores. Foco sempre em cuidado, carinho e soluções práticas.
* **CTA Padrão:** Todo Pin deve levar um link que direciona o usuário para o nosso artigo completo (ou link direto do WhatsApp Comercial se for uma oferta direta).
