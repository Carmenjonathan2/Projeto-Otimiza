### 5. **gerador_posts_gmb.py** 🤖
**O que é:** Sistema automático de geração de posts  
**Para quem:** Equipe de marketing/TI  
**Linguagem:** Python  
**Funcionalidades:**
- Gera posts de 3 tipos (dicas, produtos, depoimentos)
- Cria calendário mensal completo

**👉 Execute para gerar conteúdo automaticamente!**

---

### 6. **gerador_respostas_avaliacoes.py** 🤖
**O que é:** Sistema inteligente de respostas  
**Para quem:** Equipe de atendimento/TI  
**Linguagem:** Python  
**Funcionalidades:**
- Analisa nota e sentimento
- Gera respostas personalizadas
- Detecta contexto (atendimento, rapidez, etc)
- Identifica necessidade de follow-up
- **Economia: 5 horas/mês**

**Como usar:**
```python
from gerador_respostas_avaliacoes import GeradorRespostasAvaliacoes

gerador = GeradorRespostasAvaliacoes(
    whatsapp="31999887766",
    telefone="(31) 3333-4444"
)

resposta = gerador.gerar_resposta(
    nota=5,
    texto_avaliacao="Adorei o atendimento!"
)

print(resposta['resposta_gerada'])
```

**👉 Use para responder avaliações rapidamente!**
