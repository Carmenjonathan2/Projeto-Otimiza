## 🔲 COMO GERAR QR CODE

### Opção 1: QR Code Generator (Grátis)
1. Acesse: https://www.qr-code-generator.com
2. Cole seu link de avaliação do Google
3. Personalize:
   - Adicione logo da Otimiza no centro
   - Escolha cores da marca
   - Tamanho: 300x300 pixels (mínimo)
4. Download em alta resolução (PNG)

### Opção 2: Python (Automático)

```python
# Instalar biblioteca
pip install qrcode[pil]

# Gerar QR Code
import qrcode

# Substitua pelo seu link real
link_avaliacao = "https://g.page/r/SEU_CODIGO_AQUI/review"

qr = qrcode.QRCode(
    version=1,
    error_correction=qrcode.constants.ERROR_CORRECT_H,
    box_size=10,
    border=4,
)

qr.add_data(link_avaliacao)
qr.make(fit=True)

img = qr.make_image(fill_color="black", back_color="white")
img.save("qr-code-otimiza.png")

print("✅ QR Code gerado: qr-code-otimiza.png")
```
